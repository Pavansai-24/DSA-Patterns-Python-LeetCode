# Arrays Pattern

This folder contains LeetCode problems solved using the **Array pattern**.

## Problems Covered
- Two Sum
- Best Time to Buy and Sell Stock
- Contains Duplicate
- Maximum Subarray (Kadane’s Algorithm)
- Product of Array Except Self

## Key Concepts
- Index tracking
- Prefix sums
- Hashing with arrays
- Sliding window basics
- Kadane’s algorithm

## Goal
Build strong fundamentals in array-based problem solving, which is the base for many advanced patterns.
